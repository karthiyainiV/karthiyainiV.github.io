Marketing Sales Analysis Using SimpleLinearRegression

This portfolio project focuses on analyzing the relationship between marketing expenses and sales using simple linear regression. The objective is to gain insights into how different marketing strategies impact sales and to develop a predictive model based on the observed patterns. The project utilizes a dataset that includes information about various marketing channels such as advertising expenditure, social media engagement, email campaigns, and corresponding sales figures. The code snippet provided demonstrates the implementation of simple linear regression, a fundamental statistical modeling technique. The code begins by importing the necessary libraries and loading the dataset. It then prepares the data by splitting it into predictor (marketing expenses) and response (sales) variables. The simple linear regression model is then trained using the dataset, allowing for the estimation of the relationship between marketing expenses and sales.
