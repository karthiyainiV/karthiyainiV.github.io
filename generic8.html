NYC Apartment Data Visualization Using PowerBI

The "NYC Apartment Data Set" Portfolio Data Visualization Project aims to analyze and visualize a dataset containing information about apartments in New York City. Using Power BI, this project provides insights into various aspects of the NYC apartment market, enabling users to explore the data visually and gain valuable insights. The project involves importing the dataset into Power BI and performing data transformations and modeling to prepare the data for visualization. SQL queries are utilized to extract specific subsets of data and perform aggregations, allowing for deeper analysis and visualization.
