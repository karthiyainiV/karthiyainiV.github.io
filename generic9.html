Amazon Prime movie Data Visualization Using PowerBI

The "Amazon Prime Movie Data Visualization" project utilizes Power BI to analyze and visualize a dataset containing information about movies available on Amazon Prime. The project aims to provide insights into the movie catalog, user ratings, genres, and other relevant aspects of the Amazon Prime movie collection. The project involves importing the dataset into Power BI and performing data transformations and modeling to prepare the data for visualization. SQL queries or other data extraction techniques may be used to retrieve the necessary data from the dataset.
